# E-Commerce Web Site

## Features
- User Authentication
- Product Listings
- Shopping Cart
- Payment Integration

## Installation
1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the application: `npm start`

## Usage
- Visit `http://localhost:3000` in your browser to access the application.
